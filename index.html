Beautiful, Fast & Flexible.

It's time to bring your resume into the 21st century. This template is totally free and is meant to help you create a mobile optimized web presence. If you need help getting set up check out the instructions below, or read the documentation on github.

Getting Started

  1. Download and extract everything.
  2. Update background.jpg with a dapper picture.
  3. Edit index.html to put in your information and customize.
  4. Get it online! You can do this for free using Github Pages.
